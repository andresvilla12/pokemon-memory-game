# 🎮 Poke Memory

Este proyecto es Un juego de memoria con temática Pokémon desarrollado con React.js.El juego cuenta con 18 cartas boca abajo y el usuario intenta hacer coincidir pares de Pokémon, los datos de Pokémon se obtienen mediante PokeAPI.

Pon a prueba tu memoria encontrando todas las cartas pares.

## 🕹️ ¿Cómo jugar?

- Se muestran 18 cartas boca abajo
- Voltea dos cartas por turno
- Si hacen pareja, quedan descubiertas
- Si no coinciden, se voltean de nuevo
- ¡Encuentra todos los pares en el menor tiempo posible!

## ✨ Características

- 18 cartas con Pokémon aleatorios en cada partida
- Temporizador para medir tu tiempo
- Animación de confeti al completar el juego
- Diseño personalizado con temática nocturna Pokémon

## 🛠️ Tecnologías

- React.js
- CSS Modules
- PokeAPI
- Vite
